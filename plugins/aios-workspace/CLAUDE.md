# {{BUSINESS_NAME}} — Claude Code ワークスペース

このワークスペースは `aios-workspace` プラグインを使い、**{{BUSINESS_NAME}}** 向けにセットアップされています。

---

## このワークスペースの使い方

スキルはトピックや依頼内容に応じて自動的にトリガーされます。何ができるか分からない時は `/aios-workspace:guide` を実行してください。

---

## よく使うワークフロー

| やりたいこと | 実行するコマンド |
|---|---|
| 利用可能なスキルと現在地を確認 | `/aios-workspace:guide` |
| タスクを実行する前に計画 | `/aios-workspace:plan <やりたいこと>` |
| 保存した計画を実行 | `/aios-workspace:execute <plan-path>` |
| 未解決の質問を考える | `/aios-workspace:brainstorm <トピック>` |
| ビジネスコンテキストを追加・更新 | `/aios-workspace:update-context <内容>` |
| ワークスペースの健全性をチェック | `/aios-workspace:audit-workspace` |

---

## ワークスペース構造

| フォルダ | 用途 |
|---|---|
| `context/` | ビジネスについてClaudeが知っていること。各スキルが必要に応じて関連ファイルを読み込む。 |
| `plans/` | `plan` で作成され、`execute` で実行される日付付き計画。 |
| `outputs/` | 実行中に生み出された完成成果物。 |
| `reference/` | テンプレートと再利用可能な資料。 |
